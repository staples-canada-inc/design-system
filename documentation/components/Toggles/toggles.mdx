## Toggles

## Switches

A switch has the markup of a custom checkbox but uses the `.custom-switch` class to render a toggle switch. Switches also support the `disabled` attribute.

### Examples oF Documentation - Switches

    <div class="custom-control custom-switch">
        <input type="checkbox" class="custom-control-input" name="customSwitch" id="customSwitch1">
        <label class="custom-control-label" for="customSwitch1">Toggle this switch element</label>
    </div>
    <div class="custom-control custom-switch">
        <input type="checkbox" class="custom-control-input" name="customSwitch" id="customSwitch2">
        <label class="custom-control-label" for="customSwitch2">Or this one!</label>
    </div>

### Examples oF Documentation - Disabled Switches

    <div class="custom-control custom-switch">
        <input type="checkbox" class="custom-control-input" disabled id="customSwitch3">
        <label class="custom-control-label" for="customSwitch3">Disabled switch element</label>
    </div>
    <div class="custom-control custom-switch">
        <input type="checkbox" class="custom-control-input" disabled id="customSwitch4">
        <label class="custom-control-label" for="customSwitch4">Disabled as well</label>
    </div>
